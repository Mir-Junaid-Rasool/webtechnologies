üé® Text Effects in HTML

HTML provides various tags to add visual effects and formatting to text. Let's explore all the ways to style and format text!

Basic Text Formatting Tags

Tag Purpose Example Code Output
<b> Bold text <b>Bold</b> Bold
<strong> Important text (bold) <strong>Strong</strong> Strong
<i> Italic text <i>Italic</i> Italic
<em> Emphasized text (italic) <em>Emphasis</em> Emphasis
<u> Underlined text <u>Underline</u> Underline
<mark> Highlighted text <mark>Marked</mark> Marked
<small> Smaller text <small>Small</small> Small
<del> Deleted text <del>Deleted</del> Deleted
<ins> Inserted text <ins>Inserted</ins> Inserted
<sub> Subscript H<sub>2</sub>O H2O
<sup> Superscript x<sup>2</sup> x2
üí° Semantic vs Visual Tags:
Use <strong> instead of <b> and <em> instead of <i> for better accessibility and SEO. Semantic tags convey meaning, not just appearance!

üèóÔ∏è Various HTML Tags - DIV, SPAN, and More

1. DIV Tag - Division/Container

Purpose: Creates a block-level container to group elements together

<div style="background-color: lightblue; padding: 20px;"> <h3>This is inside a DIV</h3> <p>DIV is a block-level element.</p> </div>

This is inside a DIV

DIV is a block-level element.

2. SPAN Tag - Inline Container

Purpose: Creates an inline container for styling part of text

<p>This is <span style="color: red; font-weight: bold;">colored text</span> in a paragraph.</p>

This is colored text in a paragraph.

3. MARQUEE Tag - Scrolling Text (Deprecated but still works)

Purpose: Creates scrolling or sliding text

<marquee direction="left" behavior="scroll"> This text scrolls from right to left! </marquee>
This text scrolls from right to left!
‚ö†Ô∏è Note: Marquee is deprecated in HTML5. Use CSS animations instead for modern websites!

4. NOBR Tag - No Line Break

Purpose: Prevents text from wrapping to the next line

<nobr>This very long text will not break into multiple lines even if the container is small</nobr>

Modern Alternative: Use CSS white-space: nowrap;

5. DFN Tag - Definition

Purpose: Indicates a definition term

<p><dfn>HTML</dfn> stands for HyperText Markup Language.</p>

HTML stands for HyperText Markup Language.

6. HR Tag - Horizontal Rule

Purpose: Creates a thematic break / horizontal line

<p>Section 1</p> <hr> <p>Section 2</p>

Section 1

Section 2

7. LISTING Tag (Obsolete - For Reference Only)

Purpose: Displayed text in a fixed-width font (obsolete, use <pre> or <code>)

<pre> This is preformatted text with spaces preserved </pre>

8. Comments in HTML

Purpose: Add notes in code that won't display on the page

<!-- This is a comment --> <p>This text is visible</p> <!-- Multi-line comment for longer notes -->
üí° Best Practice: Use comments to explain complex code, mark sections, or temporarily disable code during development.

9. IMG Tag - Images

Purpose: Embeds images in the webpage

<img src="image.jpg" alt="Description of image" width="300" height="200">

Important Attributes:

  • src - Source/path to image (required)
  • alt - Alternative text for accessibility (required)
  • width - Width in pixels or percentage
  • height - Height in pixels or percentage
  • title - Tooltip text on hover

üé® Colors and Backgrounds

Ways to Specify Colors in HTML

1. Color Names

color: red; background-color: lightblue;

140+ color names available (red, blue, green, etc.)

2. Hexadecimal

color: #FF0000; background-color: #00FF00;

#RRGGBB format (most common)

3. RGB

color: rgb(255, 0, 0); background-color: rgb(0, 255, 0);

Values from 0-255 for Red, Green, Blue

4. RGBA (with transparency)

color: rgba(255, 0, 0, 0.5);

Last value (0-1) is opacity/transparency

Color Examples

red / #FF0000
blue / #0000FF
green / #00FF00
#667eea
rgb(255, 192, 203)

Background Properties

<!-- Background Color --> <body style="background-color: lightblue;"> <!-- Background Image --> <body style="background-image: url('background.jpg');"> <!-- Background with properties --> <body style=" background-image: url('bg.jpg'); background-repeat: no-repeat; background-position: center; background-size: cover; ">
