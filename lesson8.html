üéØ What are CSS Selectors?

Selectors are patterns used to select the HTML elements you want to style. They tell the browser which elements should be affected by the CSS rules.

üí° Think of selectors as:
A way to "point at" or "target" specific elements on your webpage. Just like using a pointer to highlight specific items in a list!

üìã Types of CSS Selectors

Selector Type Syntax Example Description
Universal * * { } Selects all elements
Element/Type element p { } Selects all elements of that type
Class .classname .box { } Selects elements with that class
ID #idname #header { } Selects element with that ID
Descendant parent child div p { } Selects children inside parent
Child parent > child div > p { } Selects direct children only
Grouping sel1, sel2 h1, h2 { } Applies same style to multiple
