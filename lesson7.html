üéØ What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. While HTML provides structure, CSS makes it beautiful!

üìù CSS Syntax

CSS consists of rules that tell browsers how to display HTML elements.

Basic CSS Rule Structure:
selector { property: value; property: value; }
Example:
h1 { color: blue; font-size: 24px; text-align: center; }

This heading is styled with CSS!

üîë Components:
Selector: Targets which HTML elements to style
Property: What aspect to style (color, size, etc.)
Value: The specific styling to apply
Declaration: Property + Value pair
Declaration Block: All declarations inside { }

üìå Three Ways to Apply CSS

1. Inline CSS

Styles applied directly to HTML elements using the style attribute.

<p style="color: red; font-size: 18px;">This is inline CSS</p>

This is inline CSS

‚úÖ Pros: Quick for single elements, highest priority
‚ùå Cons: Hard to maintain, mixes HTML and CSS, no reusability

2. Internal CSS

Styles defined in the <style> tag within the HTML document's <head>.

<head> <style> p { color: green; font-size: 16px; } </style> </head>
‚úÖ Pros: Styles specific to one page, good for single-page sites
‚ùå Cons: Increases page size, can't be reused across multiple pages

3. External CSS (Best Practice!)

Styles defined in a separate .css file and linked to HTML.

<!-- In HTML file --> <head> <link rel="stylesheet" href="styles.css"> </head> /* In styles.css file */ p { color: purple; font-size: 18px; }
‚úÖ Pros: Clean separation, reusable across multiple pages, easier maintenance, cacheable
‚ùå Cons: Extra HTTP request (minimal impact)
