üìã What are HTML Tables?

Tables are used to organize and display data in rows and columns. They're perfect for displaying structured information like schedules, pricing, comparisons, and statistical data.

üìä Data Organization

Present information in a structured, easy-to-read format

üîç Easy Comparison

Compare multiple items or values side by side

üìà Visual Clarity

Make complex data easier to understand and analyze

üí° Important Note:
Tables should be used for tabular data only, NOT for page layout! In the past, tables were misused for layout, but modern CSS (Grid and Flexbox) should be used for page layouts instead.

üéØ Key Takeaways

Table Basics:
‚úÖ Use <table> to create tables
‚úÖ <tr> creates rows, <td> creates data cells, <th> creates header cells
‚úÖ Use semantic elements: thead, tbody, tfoot
‚úÖ Add captions with <caption> tag

Colspan & Rowspan:
‚úÖ colspan merges cells horizontally (across columns)
‚úÖ rowspan merges cells vertically (across rows)
‚úÖ Can combine both in the same table
‚úÖ Always count total cells carefully

Best Practices:
‚úÖ Use tables for tabular data only, not layouts
‚úÖ Use CSS for styling instead of HTML attributes
‚úÖ Include headers (<th>) for accessibility
‚úÖ Make tables responsive for mobile devices
‚úÖ Use border-collapse: collapse for clean borders
