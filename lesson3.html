‚ö†Ô∏è Important Best Practices:

1. Always use alt attribute: Essential for accessibility and SEO
2. Specify width and height: Prevents page layout shift
3. Use descriptive alt text: Describe what's in the image
4. Optimize image size: Compress images before uploading
5. Use correct formats: JPEG for photos, PNG for graphics/transparency, SVG for logos

Image Alignment (Using CSS Style)

Modern Way: Using CSS Style Attribute
<!-- Left Aligned --> <img src="photo.jpg" alt="Photo" style="float: left; margin-right: 10px;"> <!-- Right Aligned --> <img src="photo.jpg" alt="Photo" style="float: right; margin-left: 10px;"> <!-- Centered --> <img src="photo.jpg" alt="Photo" style="display: block; margin: 0 auto;">

üìù Practice Exercises

Exercise 1: Create a Recipe Page

Build an HTML page for your favorite recipe with:

  • Ordered list for step-by-step instructions
  • Unordered list for ingredients
  • Definition list for cooking terms/techniques
  • At least one image of the dish

Exercise 2: Course Syllabus Page

Create a syllabus page for any subject with:

  • Ordered list of modules (1, 2, 3...)
  • Nested unordered lists for topics under each module
  • Definition list for key terms
  • Images for visual aids (charts, diagrams)

Exercise 3: Product Catalog

Design a simple product catalog with:

  • 3-5 product images with proper attributes
  • Unordered list of features for each product
  • Definition list for specifications
  • Use different image sizes and alignments

Challenge: FAQ Page with Images

Create a Frequently Asked Questions page:

  • Use definition lists for Q&A pairs
  • Nested lists for multi-part answers
  • Add relevant images with proper alt text
  • Use different list types (OL, UL, DL) appropriately
  • Include at least one nested list (3 levels deep)
