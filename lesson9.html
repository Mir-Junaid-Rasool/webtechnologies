Color Values in CSS

CSS supports multiple ways to define colors:

Method Syntax Example Use Case
Color Names name red, blue, green Quick & simple
Hexadecimal #RRGGBB #FF5733 Most common
RGB rgb(r, g, b) rgb(255, 87, 51) Precise control
RGBA rgba(r, g, b, a) rgba(255, 87, 51, 0.5) With transparency
HSL hsl(h, s%, l%) hsl(14, 100%, 60%) Intuitive adjustments
HSLA hsla(h, s%, l%, a) hsla(14, 100%, 60%, 0.5) HSL + transparency

Property Values Description

word-spacing px, em Space between words

‚úÖ text-shadow creates shadow effects
