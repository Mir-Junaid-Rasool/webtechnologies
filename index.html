üìú The Evolution of HTML & The Web

üåç Did You Know?
The World Wide Web was created to share scientific information between researchers. Today, it connects over 5 billion people worldwide and hosts over 1.7 billion websites!
1989

üåü The Birth of the Web

Tim Berners-Lee at CERN proposes a global hypertext project. His vision: a system where documents could link to each other across computers worldwide.

Key Innovation: Hypertext Transfer Protocol (HTTP)

1991

üìÑ HTML is Born

First HTML specification with just 18 elements! Simple tags like <p>, <h1>, <a>, and <img> formed the foundation.

Fun Fact: The first website (info.cern.ch) is still accessible today!

1993

üñºÔ∏è Images Arrive!

The <img> tag is introduced, allowing images to be embedded in web pages. The web suddenly becomes visual!

Also: Mosaic browser makes the web accessible to non-technical users

1995

üé® The Styling Era Begins

HTML 2.0 becomes official standard. JavaScript is created in just 10 days by Brendan Eich, bringing interactivity to web pages!

CSS is also proposed to separate content from presentation

1997-1999

üí™ HTML Matures

HTML 4.0 brings support for stylesheets, scripting, frames, and better accessibility. The web becomes more powerful and accessible.

CSS2 released, giving designers unprecedented control

2000-2006

üîÑ XHTML: The XML Experiment

XHTML attempts to make HTML more strict and XML-compliant. While not widely adopted, it teaches important lessons about code quality.

Web 2.0 movement: Social media and user-generated content explode

2008-2014

üöÄ HTML5 Revolution

HTML5 brings native video/audio, canvas for graphics, semantic elements, offline storage, and more! No more Flash needed.

Key features: <video>, <audio>, <canvas>, <svg>, localStorage

2016-Present

üåê Living Standard

HTML becomes a Living Standard, continuously evolving with new features like Web Components, PWAs, and WebAssembly.

Focus: Mobile-first, accessibility, performance, and security

üíª HTML Basics: Building Your First Page

ü§î What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure and content of a web page using a series of elements called tags.

üìÑ
Markup Language

Not a programming language - it marks up content to give it structure

üèóÔ∏è
Structure

Defines headings, paragraphs, links, images, and more

üåê
Universal

Works on all browsers and devices

üì±
Foundation

The backbone of every website

üìã Basic HTML Document Structure

Every HTML document follows this fundamental structure:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Web Page</title> </head> <body> <h1>Welcome to My Website</h1> <p>This is my first paragraph.</p> <!-- This is a comment --> </body> </html>

üîç Understanding the Parts

<!DOCTYPE html>
Tells the browser this is an HTML5 document. Always put this first!
<html>
The root element that contains all other HTML elements
<head>
Contains metadata, title, links to CSS, and other information not displayed on the page
<body>
Contains all the visible content: text, images, links, etc.

üè∑Ô∏è Tags and Elements

HTML uses tags to create elements:

Opening Tag

<tagname>

Marks the start of an element

Content

The actual content

Text, images, or other elements

Closing Tag

</tagname>

Marks the end of an element

‚ú® Common HTML Tags

<!-- Headings (h1 is largest, h6 is smallest) --> <h1>Main Heading</h1> <h2>Subheading</h2> <!-- Paragraph --> <p>This is a paragraph of text.</p> <!-- Line Break --> Line 1<br>Line 2 <!-- Text Formatting --> <strong>Bold text</strong> <em>Italic text</em> <u>Underlined text</u> <!-- Link --> <a href="https://example.com">Click here</a> <!-- Image --> <img src="image.jpg" alt="Description">

üéØ Attributes

Attributes provide additional information about elements:

<tag attribute="value">content</tag> <!-- Examples --> <a href="page.html" target="_blank">Link</a> <img src="photo.jpg" alt="Photo" width="300"> <p class="intro" id="first">Text</p>
üí° Pro Tip: Attributes are always specified in the opening tag and come in name="value" pairs. Common attributes include: id, class, src, href, alt, width, height, style, title
